This folder contains screenshots that document the troubleshooting and configuration steps performed in the Enterprise Active Directory Lab.

Each image corresponds to a step in the lab including:

• Domain Controller deployment in Microsoft Azure  
• Active Directory user and policy configuration  
• Password and account lockout policy testing  
• RDP authentication troubleshooting  
• SMB file share access verification  
• DNS lookup validation  
• Network packet analysis using Wireshark  

These screenshots support the documentation provided in the main README and demonstrate hands-on experience performing common IT Support and Service Desk tasks in a Windows enterprise environment.
